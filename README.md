# akirahanada.github.io

Personal ePortfolio of **Akira Hanada** — Product Designer &amp; UX Designer.

Live site: **https://akirahanada.github.io/**

Built as a fast, responsive, single-page static site (HTML, CSS, vanilla JS) and
deployed with GitHub Pages.

## Contents
- `index.html` — page structure and content
- `styles.css` — design system, layout, and responsive rules
- `script.js` — mobile navigation and scroll animations

## Featured work
1. **Rakuten Ichiba Cart Redesign** — mobile-first, research-driven checkout redesign.
2. **Hinode Bank Mobile Banking Wireframes** — low-fidelity Balsamiq prototype and navigation map.

Created for CSYE 7280 — User Experience Design &amp; Testing, Northeastern University.
